概览

您好! 金矢科技作为一家以技术服务为导向的企业，在研发运维上，积累了几年经验。从云计算到企业IT系统，从移动互联网到物联网，我们都有人才、产品。帮助您借助信息技术改善效益，是我们一直努力实现的目标！

金矢这个名字，取自中国古典《菜根谭》中的“金须百炼，矢不轻发”。作为一家开放的公司，大量的采用开源技术，同时独立自主的打造自身产品，用服务和口碑去推广品牌。在为客户提供优秀服务、产品的同时，也为努力为开源社区做推广、贡献。

一个企业，或类比为机构，从供应、生产到发布产品、服务大体如上图。金矢科技的接包服务科技能打通每一节，从供应商到基础设施，可以用RFID／条码／网上支付，从基础设施到运营可以使用ERP和BPM构建业务流程做资产管理、人员管理，从运营通过开发移动客户端／微信公众号让消费者多一条便捷的途径得到服务／产品。
